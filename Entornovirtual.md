  ![alt text](https://github.com/KarenHernandez08/Ejercicios_Python/blob/main/Imagenes/carpeta.PNG)

  ![alt text](https://github.com/KarenHernandez08/Ejercicios_Python/blob/main/Imagenes/virtualenv.PNG)
  
  #### Creamos nuestro entorno virtual env asi `virtualenv entornovirtual` donde "entornovirtual" es mi nuevo directorio
  ![alt text](https://github.com/KarenHernandez08/Ejercicios_Python/blob/main/Imagenes/entorno.PNG)
  
   #### Activamos nuestro entorno virtual con el siguiente comando `.\entornovirtual\Scripts\activate` 
  ![alt text](https://github.com/KarenHernandez08/Ejercicios_Python/blob/main/Imagenes/activar.PNG)
  
   #### Ejecutamos este comando para ver las bibliotecas instaladas `pip freeze` en este caso como aun no instalamos ninguna no debemos tener respuesta
